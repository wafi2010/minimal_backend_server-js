# 🚀 Minimal Node.js & Express.js Backend Server

এটি একটি অত্যন্ত সাধারণ (Minimal) ব্যাকএন্ড সার্ভার প্রজেক্ট। এই প্রজেক্টটি মূলত তৈরি করা হয়েছে যারা একদম নতুন এবং শিখতে চাচ্ছেন কীভাবে **Node.js** এবং **Express.js** ব্যবহার করে একটি সার্ভার তৈরি করতে হয়।

## 📌 প্রজেক্টের বৈশিষ্ট্য (Features)
- **Node.js**: জাভাস্ক্রিপ্ট রানটাইম এনভায়রনমেন্ট।
- **Express.js**: ব্যাকএন্ড ফ্রেমওয়ার্ক হিসেবে ব্যবহৃত।
- **Nodemon**: ডেভেলপমেন্টের সময় কোড পরিবর্তন করলে সার্ভার অটো-রিস্টার্ট করার জন্য।
- **Basic Routing**: একটি সিম্পল GET রুট সেটআপ।

- [Node.js](https://nodejs.org/) (LTS Version recommended)
- একটি কোড এডিটর (যেমন: [VS Code](https://code.visualstudio.com/))
